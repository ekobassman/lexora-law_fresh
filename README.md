# Lexora

## Environment

Copy `.env.example` to `.env` and set:

- `VITE_SUPABASE_URL` – Supabase project URL (default project: `https://ttzykwwaruxcstftanae.supabase.co`)
- `VITE_STORAGE_BUCKET` – Storage bucket for uploads (default: `documents`)
- `VITE_SUPABASE_ANON_KEY` – Anon key from Supabase Dashboard (Project Settings → API)

Do not commit `.env` or real API keys.
